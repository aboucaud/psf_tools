      Input image

    • sigma –

      Width of the weighting Gaussian in arcsec

    • pixscl –

      Pixel scale of the image to scale the Gaussian function

    • n_iter –

      Number of iterations of the weighting (default 3)

    Returns:

    • mu_x, mu_y –

      Coordinates of the cendroid in the image plane

    circularize(image, n_iter=14, interp_order=2, log=False) ¶

    Circularize an image.

    Successively rotate the image and add it to itself with angles decreasing by a factor 2 every iteration. This is numerically equivalent to computing the average over 2^n rotations where n is the number of iterations. THe default choice of n_iter = 14 hence produces an image invariant under rotations of any angle that is a multiple of 360/2^14 = 360/16384 ~ 0.022 deg

    Parameters:

    • image –

      Image to circularize

    • n_iter –

      Number of iterations in the process (default 14)

    • interp_order –

      Spline interpolation order in the rotation * 0 : linear * 1 : bilinear * 2 : bicubic (default)

    Returns:

    • Circularized image –
